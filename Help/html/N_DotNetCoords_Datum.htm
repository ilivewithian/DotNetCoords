Classes

Datum
The Datum class represents a set of parameters for describing a particular datum, including a name, the reference ellipsoid used and the seven parameters required to translate co-ordinates in this datum to the WGS84 datum.
DatumT
Generic datum class represents a set of parameters for describing a particular datum, including a name, the reference ellipsoid used and the seven parameters required to translate co-ordinates in this datum to the WGS84 datum.
